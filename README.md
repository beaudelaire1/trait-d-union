# Trait d'Union - Agence Web en Guyane

Le site de mon agence web en Guyane, spécialisée dans le développement web sur mesure.

## À propos

Trait d'Union est une agence web basée en Guyane française, offrant des services de développement web, design UI/UX, applications web sur mesure et optimisation SEO.

## Démarrage

Pour visualiser le site localement :

1. Clonez ce dépôt
2. Ouvrez `index.html` dans votre navigateur web

Aucune installation ou dépendance n'est requise - il s'agit d'un site statique simple.

## Structure du projet

- `index.html` - Page principale du site
- `styles.css` - Styles CSS
- `script.js` - JavaScript pour les interactions

## Fonctionnalités

- Design responsive adapté aux mobiles
- Navigation fluide avec défilement doux
- Formulaire de contact
- Présentation des services
- Section à propos
- Animations au défilement

## Déploiement

Ce site peut être déployé sur n'importe quelle plateforme d'hébergement statique :
- GitHub Pages
- Netlify
- Vercel
- Ou tout serveur web traditionnel
